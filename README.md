# Блок 4: Анализ данных и автоматизация на Python

В этом репозитории собраны скрипты и jupyter-ноутбуки, демонстрирующие навыки программирования, обработки данных и построения графиков на языке Python.

## 🛠️ Что было сделано:
- Проведен разведочный анализ данных (EDA) с использованием библиотек `pandas` и `numpy`.
- Выполнена предобработка данных: обработка пропусков, удаление дубликатов, изменение типов данных и фильтрация.
- Построены графики и диаграммы для поиска зависимостей с помощью `matplotlib` и `seaborn`.
- Автоматизированы рутинные задачи по группировке и агрегации статистических показателей.

## 📁 Структура файлов в репозитории:
- `analysis_notebook.ipynb` — Jupyter Notebook с кодом, графиками и пошаговыми комментариями.
- `requirements.txt` — список библиотек, необходимых для запуска кода.
- `README.md` — описание проекта и ключевых выводов исследования.
